# ChartCraft

ChartCraft คือ Drag & Drop Online Dashboard Builder Platform

## เป้าหมาย
- Login / CMS / Dashboard Builder
- Drag & Drop widget
- Chart Library
- Excel / CSV / Manual Data / Google Sheet
- Publish เป็น URL
- Export เป็น HTML file
- Storage / Security / Performance ตั้งแต่ต้น

## วิธีใช้กับ Codex
ให้ Codex อ่าน `AGENTS.md`, `AI_CONTEXT.md`, `PROJECT_RULES.md`, `PRODUCT_REQUIREMENTS.md`, `ROADMAP.md` ก่อน แล้วใช้ prompt ใน `templates/CODEX_START_PROMPT.md`
