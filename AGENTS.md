# AGENTS.md

กฎสูงสุดสำหรับ AI Coding Agent ทุกตัวในโปรเจกต์ ChartCraft

## ภาษา
ต้องสื่อสารกับผู้ใช้เป็นภาษาไทย 100% รวมถึงวิเคราะห์งาน ถามตอบ รายงานผล TODO Checklist Bug report และแผนทดสอบ

ยกเว้น Source code, variable, function, class, component, database schema, API endpoint, package name และ command line ให้ใช้ภาษาอังกฤษตามมาตรฐาน

## ก่อนเริ่มทุก Phase
ต้องถามผู้ใช้ก่อนเสมอว่า “อนุมัติให้เริ่ม Phase นี้หรือไม่?”

## ก่อนเขียนโค้ด
ต้องอธิบายว่าจะทำอะไร แก้ไฟล์ไหน กระทบ database/storage/security/performance หรือไม่ และจะทดสอบอย่างไร

## หลังทำงาน
ต้องสรุปผลเป็นภาษาไทย และอัปเดตเอกสารที่เกี่ยวข้อง
